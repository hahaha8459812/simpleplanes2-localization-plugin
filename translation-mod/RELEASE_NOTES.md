# v0.1.2

本次更新继续扩大汉化覆盖，并优化运行时翻译性能。

## 主要更新

- 新增和完善多类零件翻译：螺旋桨发动机、喷气发动机、涡扇发动机、VTOL 发动机、喷气发动机整流罩、主旋翼、直升机尾桨、传动系统、变速箱、变速器、差速器、方向盘、踏板、操纵杆、按钮、开关、握把和座舱盖等。
- 补充发动机/旋翼属性翻译：加力燃烧室、燃烧室温度、涵道比、压缩比、燃油消耗、推力、反推、喷口样式、进气锥样式、桨叶直径、桨叶厚度、桨距控制、油门调速器、尾桨输入等。
- 补充发动机相关悬浮说明，包括涵道比、燃烧室温度、燃油消耗、海平面静推力等说明文本。
- 补充座舱控件与输入控制相关翻译，包括按钮动作、握把控制方案、激活组、武器开火、发射干扰物、起落架、油门和配平等。
- 补充新建作品确认框和更多设计器缺漏文本。

## 体验与性能优化

- 优化文本扫描策略：平时低频扫描，用户点击、按键、滚轮或场景切换后短时间高频扫描，降低持续轮询开销。
- 缓存已处理文本，减少重复翻译和重复字体处理。
- 修复滑条快速拖动或点击跳转数值后翻译偶尔消失的问题。
- 鼠标松开后延迟 0.1 秒主动扫描一次，改善滑条联动刷新后的翻译恢复速度。
- 快捷键改为 `F1` 开关翻译、`F2` 刷新翻译。

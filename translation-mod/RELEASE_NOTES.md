# v0.1.3

本次更新主要改进游戏内汉化控制体验，并补充机身编辑器相关翻译。

## 主要更新

- 增加游戏内汉化浮窗面板，可直接重载翻译、临时开关汉化和打开设置。
- 浮窗面板改为中文显示，并加载 Windows 中文字体，降低按钮文字显示异常的概率。
- 支持在面板中启用/关闭快捷键，并自定义重载与开关汉化快捷键。
- 浮窗位置会写入配置文件；拖动面板后松开鼠标即可保存，下次启动继续使用上次位置。

## 修复与体验优化

- 修复关闭汉化后再次开启不立即恢复的问题。
- 修复早期浮窗鼠标拦截方案导致游戏界面被全局拦截的问题；当前版本不再使用透明 Canvas 遮罩。
- 调整鼠标事件处理，仅消费 IMGUI 浮窗区域内的鼠标事件，避免影响浮窗外的游戏操作。

## 翻译补充

- 补充机身编辑器相关文本：`Per Corner`。
- 将 `Rise` 调整为“垂直偏移”。
- 将 `Run` 调整为“水平偏移”。
